# IsStress

## AI-Powered Stress Detection System

### Project Description

IsStress is a healthcare-focused stress detection web application.

The user answers a 15-question quiz related to sleep, work/study pressure, emotions, social life, and physical symptoms.

The answers are analyzed using an XGBoost Machine Learning model which predicts the user's stress level.

The application provides:

- Stress Level
- Stress Score (Out of 100)
- SHAP-based Explanation
- Personalized Recommendations
